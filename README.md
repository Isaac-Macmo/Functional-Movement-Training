
# Functional Movement Training - Static Website (multi-page)

This is a simple static website (HTML/CSS/JS) with four pages:
- index.html (Home)
- about.html (About)
- pricing.html (Pricing)
- contact.html (Contact / Apply)

## How to upload to GitHub (beginner-friendly)
1. Unzip the downloaded `functional_movement_site.zip` on your computer.
2. Go to https://github.com and open the repository you want to use (or create a new repo).
3. Click **Add file â†’ Upload files**.
4. Drag **all files from the unzipped folder** into the upload area (make sure files are in the root, not inside another folder).
5. Scroll down and click **Commit changes**.
6. Go to Vercel and your project should auto-deploy. If not, click **Deploy** or **Redeploy** in the Vercel project dashboard.

## Edit contact email
Open `contact.html` and replace `youremail@example.com` with your real email.
